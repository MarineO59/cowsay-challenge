# Cowsay Challenge

Challenge Node.js utilisant le package cowsay et les variables d'environnement.

## Installation

1. Cloner le projet
2. Installer les dépendances :
```bash
npm install
```
3. Créer un fichier `.env` avec :
```
NAME=YOUR_NAME
CAMPUS=YOUR_CAMPUS
```

## Utilisation
```bash
node --env-file=.env index.js
```

## Résultat

Une vache affiche votre nom et campus !
